# WorkingWithFiles

Functions to find and list files


## Install from PYPI

The homepage in pipy is https://pypi.org/project/WorkingWithFiles/

```bash
pip install --upgrade WorkingWithFiles
```

## Install from source
Installing `WorkingWithFiles` program

```bash
git clone https://github.com/trucomanx/WorkingWithFiles.git
cd WorkingWithFiles
pip install -r requirements.txt
cd src
python -m build
pip install dist/workingwithfiles-*.tar.gz
```

## Uninstall

```bash
pip uninstall WorkingWithFiles
```
